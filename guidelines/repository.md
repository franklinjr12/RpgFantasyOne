# Notes relevant on how this repository works
It uses Go programming language with raylib for any game related systems that are not custom to this app.
For any reference check https://www.raylib.com/cheatsheet/cheatsheet.html which is the reference of C++ calls but can be easily translated into Go calls as well.
The app is Windows OS only and this will not change.
In case you ever need more details on the game systems check the markdown files under guidelines folder.
